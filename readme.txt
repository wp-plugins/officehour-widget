== Description ==

Le Widget OfficeHour permet d'afficher sur son blog WordPress un profil d'expert OfficeHour.
Le Widget est personnalisable (couleurs, dimensions et textes) et s'installe en quelques clics.

== Installation ==

Comment installer le plugin OfficeHour Widget :

1. Installez le automatiquement en utilisant le menu Plugins > "Ajouter un Nouveau" > Rechercher "OfficeHour Widget" (ou ajouter le dossier complet officehour-widget dans le répertoire /wp-content/plugins/)
2. Activez le plugin
3. Utilisez et configurez le depuis le menu "Apparence" > "Widgets"
4. Deplacer et glisser le widget "OfficeHour Widget" dans votre sidebar (ou ailleurs)
5. Configurer le widget en renseignant votre nom d'utilisateur OfficeHour

== Frequently Asked Questions ==

= Quel est mon nom d'utilisateur OfficeHour ? =

Rendez-vous dans votre compte OfficeHour pour obtenir votre nom d'utilisateur : [Mon Compte](http://www.officehour.fr/expert/profile/description)
